Mantenimiento de Una Computadora "Laptop"


En la presente práctica se llevará a cabo un mantenimiento de una laptop, con el objetivo de garantizar su óptimo rendimiento. Este mantenimiento implicará una limpieza profunda que abarca diversos pasos detallados, así como la consideración de cuidados específicos para asegurar un óptimo resultado. Es esencial realizar una limpieza completa de los equipos de trabajo, como las laptops, con el fin de preservar su eficiencia y prolongar su vida útil. Durante el procedimiento, se seguirán pasos recomendados para llevar a cabo la limpieza, los cuales incluyen la remoción de polvo y suciedad de las superficies internas y externas de la laptop. Se prestará especial atención a áreas de difícil acceso, como los ventiladores y las ranuras de ventilación, que pueden acumular una cantidad de residuos y afectar el rendimiento de la Laptop.


Materiales para el Mantenimiento de una Laptop


  • Aire comprimido
  • Limpia Contactos o Alcohol Isopropílico
  • Pasta térmica
  • Trapo de Limpieza o Toalla Absorbente (Papel Mayordomo)
  • Kit de destornilladores

Instalación del Sistema Operativo Linux en VirtualBox


La instalación de Linux en VirtualBox permite crear un entorno virtualizado para explorar y utilizar sistemas operativos basados en Linux en tu computadora o Laptop. VirtualBox ofrece una plataforma segura y flexible para probar diferentes distribuciones de Linux, como Ubuntu, Manjaro, entre otros, sin afectar tu sistema operativo principal. Este proceso implica crear una máquina virtual dentro de VirtualBox y luego instalar Linux dentro de esa máquina virtual siguiendo pasos simples y guiados.

Configuración de Conexiones de Redes


Configurar conexiones de redes inalámbricas y por cable entre sistemas operativos como Windows y Linux es fundamental para la comunicación fluida entre entornos mixtos. Esto implica establecer configuraciones de red adecuadas, como asignación de direcciones IP, configuración de adaptadores de red virtuales en entornos de virtualización. Este proceso facilita la transferencia de datos y recursos compartidos entre diferentes sistemas, promoviendo la compatibilidad entre redes por cables/inalámbricas.

Comandos para uso de Consola en Ubuntu


  • sudo Es un comando que permite realizar acciones como administrador
  • ls Se usa para ver el contenido de un directorio
  • touch Sirve para crear un archivo vacio
  • cp Se utiliza para copiar un archivo
  • nano Cumple la función de editar un archivo
  • rm Se usa para eliminar un archivo
  • mv Tiene la función de mover un archivo a otra carpeta
  • cd Tiene la función de moverse entre carpetas, si se coloca solo se sale de la carpeta actual
  • mkdir Sirve para crear un directorio
  • pwd Muestra la ruta de la carpeta actual
  • apt-get update Actualiza la lista de paquetes
  • apt-get upgrade Actualiza los paquetes instalados
  • apt-get install Instala un paquete
  • apt-get remove Desinstala un paquete
  • apt-get purge Desinstala un paquete y sus archivos de configuración
  • apt-get autoremove Elimina paquetes que no se usan
  • clear Limpia la consola
  • exit Cierra la consola
